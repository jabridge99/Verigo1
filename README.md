# Smart E-Waste Recycling Platform

This repository currently contains the **system architecture and proposed implementation structure only** for the smart e-waste recycling MVP.

## Current status
- This revision is intentionally **documentation-first**.
- No backend, frontend, IoT worker, or infrastructure code is included yet.
- The folder and file paths referenced in the planning documents describe the **target implementation layout**, not the current repository contents.

## Planning documents
- `docs/system-architecture.md` — target architecture for IoT ingestion, backend modules, data, security, deployment, and future scale.
- `docs/folder-structure.md` — proposed repository layout and rationale for the future implementation phase.
- `backend/README.md` — backend boundary and module planning notes.
- `frontend/README.md` — frontend flow and structure planning notes.
- `infra/README.md` — infrastructure planning notes.
- `.env.example` — configuration contract placeholder for the future implementation.
